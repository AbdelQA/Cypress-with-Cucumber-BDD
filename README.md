# Cypress Tests using Cucumber BDD Framework with Gherkin Statements
(NOTE: Work In Progress)

## Installing:

` npm install `

## Running Tests (Headless Mode):

`npx cypress run`

## Running Tests (Headed Mode, Cypress UI):

`npx cypress open`
